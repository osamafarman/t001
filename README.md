# ثلاثية — موقع متعدد الصفحات

## الملفات
- `index.html` — الصفحة الرئيسية
- `calculator.html` — الآلة الحاسبة
- `game.html` — لعبة الذاكرة
- `flags.html` — أعلام الدول

## طريقة الرفع على GitHub Pages

### الخطوة 1: إنشاء حساب GitHub
- اذهب إلى https://github.com وسجّل حساباً مجانياً

### الخطوة 2: إنشاء Repository جديد
- اضغط على زر "+" ثم "New repository"
- اسمه مثلاً: `my-site`
- اتركه Public
- اضغط "Create repository"

### الخطوة 3: رفع الملفات
- داخل الـ repository اضغط "uploading an existing file"
- ارفع الملفات الأربعة (index.html, calculator.html, game.html, flags.html)
- اضغط "Commit changes"

### الخطوة 4: تفعيل GitHub Pages
- اذهب إلى Settings > Pages
- في "Source" اختر "Deploy from a branch"
- Branch: main / root
- اضغط Save

### الخطوة 5: رابط موقعك
بعد دقيقتين يصبح موقعك متاحاً على:
`https://اسم-المستخدم.github.io/my-site/`
